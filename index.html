Methodology

The model combines team-strength variables, context adjustments, and a tournament simulation layer. Live mode uses the latest JSON data generated by GitHub Actions. Scenario mode lets users test assumptions without changing the official forecast.

adjustedStrength = weightedInputs
  − injuryRisk × injuryImpact
  − pathDifficulty × pathImpact
  − travelBurden × travelImpact
  + climateFit × climateImpact
  + marketPrior × marketInfluence

Limitations

  • This is a probabilistic model, not a guarantee.
  • Free injury and squad sources may be incomplete or delayed.
  • The official third-place knockout assignment is approximated in the simulator.
  • Market data is used only as a consensus signal, not betting advice.
